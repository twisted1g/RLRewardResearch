## Об исследовании

Работа посвящена сравнительному анализу алгоритмов **Deep Q-Network (DQN)** и **Advantage Actor-Critic (A2C)** с четырьмя различными функциями награды применительно к торговле криптовалютной парой **BTC/USDT** на часовых данных.

**Ключевой вывод:** функция награды оказывает принципиальное влияние на результаты — разброс Total Return на тестовой выборке достигает ~1.3 пункта для DQN (от −0.03 до 1.28). DQN с LogReturn показал наилучший результат (Total Return = 1.28), тогда как A2C демонстрирует более стабильные, но консервативные результаты (0.23–0.55).


## Торговая среда

**Данные:** BTC/USDT, таймфрейм 1h, источник — Binance.
- Обучение: январь 2022 — декабрь 2023
- Тестирование: 2024 год

**Пространство действий (дискретное):**
| Действие | Описание |
|----------|----------|
| `0` | Flat |
| `1` | Long |
| `2` | Short |

**Вектор состояния (6 признаков):**
| Признак | Описание |
|---------|----------|
| log-return | Логарифмическая доходность |
| volatility | Волатильность |
| z-score price | Z-score цены |
| z-score volume | Z-score объёма |
| EMA diff | Разность скользящих средних |
| position | Текущая позиция агента |

Реализованы ограничения на максимальную длительность позиции и допустимую просадку.

## Функции награды

| Название | Формула | Описание |
|----------|---------|----------|
| `LogReturn` | $r_t = \log(V_t / V_{t-1})$ | Логарифмическая доходность портфеля |
| `ReturnReward` | $r_t = (V_t - V_{t-1}) / V_{t-1}$ | Простая доходность портфеля |
| `SharpeReward` | $r_t = \mu_t / (\sigma_t + \varepsilon)$ | Отношение доходности к риску |
| `DrawdownAwareReward` | $r_t = r_t^{base} - \lambda \cdot DD_t$ | Базовая награда со штрафом за просадку |

| Алгоритм | Функция награды | Total Return | Max Drawdown |

| DQN | DrawdownAware | 0.21 | −0.38 |

| A2C | DrawdownAware | 0.23 | −0.28 |

### Ключевые наблюдения

- **DQN + LogReturn** — единственная конфигурация с существенным ростом портфеля (1.28) при умеренной просадке (−0.29).
- **DQN чувствителен** к выбору функции награды: результаты варьируются от −0.03 до 1.28.
- **A2C стабилен**, но консервативен: все конфигурации дают положительную доходность (0.23–0.55), однако ни одна не превышает доходность базового актива.
- **SharpeReward** по-разному влияет на алгоритмы: для A2C — умеренный результат (0.24), для DQN — отрицательная доходность (−0.03).
- **DrawdownAwareReward** снижает просадку A2C до −0.28, но ограничивает доходность. У DQN приводит к нестабильному обучению.

## Данные

## Быстрый старт

```python
import pandas as pd
from env import MyTradingEnv, DrawdownAwareReward
from agents.train_dqn import train_dqn, DQNConfig
from agents.experiment import Experiment

df = pd.read_csv("data/raw/binance_BTCUSDT_1h.csv")

env = DrawdownAwareReward(df, max_steps=2000)

config = DQNConfig(total_timesteps=50_000)
exp = Experiment("dqn-drawdown")
train_dqn(env, config, exp)
```
